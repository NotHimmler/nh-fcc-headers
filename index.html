nh-fcc-headers

This is a micro-service API developed as part of FreeCodeCamp's Full-Stack/Back-End curriculum.

It uses the Node.js runtime and Express.js to provide the backend functionality.

Going to http://nh-fcc-headers.herokuapp.com/api will return a JSON object with your IP Address, Operating System details and display language.

Example Result

{"ipaddress":"10.0.2.2","software":"Windows NT 10.0; WOW64","language":"en-US"}
